# Brijesh Chinai — Biography

A single-page biography website for Brijesh Chinai, Founder & Chairperson of the BRIC Group of Companies.

## Files

```
index.html          The website (open this)
assets/brijesh.jpg  Portrait image
```

## Publish on GitHub Pages

1. Create a new repository on GitHub.
2. Upload **all files in this folder** (keep `index.html` at the top level and the `assets` folder alongside it).
3. Go to **Settings → Pages**.
4. Under **Source**, choose the `main` branch and the `/ (root)` folder, then **Save**.
5. Your site goes live at `https://<your-username>.github.io/<repo-name>/` within a minute or two.

## Editing

- **Email address:** open `index.html`, find `mailto:hello@brijeshchinai.com`, and replace with the real address.
- **Location line:** search for `Mumbai, India`.
- **Photo:** replace `assets/brijesh.jpg` with a new image of the same name (portrait / 2:3 works best).

Fonts load from Google Fonts, so an internet connection is needed for the exact typography.
